# Настройка параметров езды

**Сейчас езда в прототипе** задаётся **`CarControl`** и камерой **`FollowCamera`**; **топливо** — **`FuelSystem`** (связан с **`CarControl`**) и UI **`FuelUI`**. Полный список полей: **[Параметры транспорта](Параметры%20транспорта.md)**.

**`CarStats`:** **`FuelSystem`** читает **`maxFuel`** и **`maxSpeedKmh`**. **`CarControl`** ассет для силы мотора и угла руля **не читает** — эти параметры по-прежнему крутятся в Inspector у **`CarControl`**.

---

## CarControl — что крутить в первую очередь

| Поле | Зачем крутить |
| ---- | ------------- |
| **Engine Power** | Разгон и «толчок» заднего хода. Низко — машина ленивая; высоко — резкий старт и переворот на кочках. |
| **Turn Speed** | Максимальный угол передних колёс (градусы). Выше — острее поворот на малых скоростях. |
| **Turn Smoothness** | Как быстро руль догоняет ввод. Ниже — вальяжнее; выше — отзывчивее. |
| **Use Drift / Drift Amount / Drift Sideways Grip Min / Drift Speed Ref Ms** | Сила и ранность заноса (боковая жёсткость шин). Подробно в [Параметры транспорта](Параметры%20транспорта.md). |
| **Steer Input Deadzone / Yaw Straighten Strength / Steer Return Multiplier** | Поведение после отпускания руля: меньше «залипания» поворота или больше инерции. |
| **Fuel System** | Ссылка на **`FuelSystem`** (если пусто — ищется на том же GameObject). Без компонента машина едет без расхода топлива. |

Массивы **Wheels** и **Wheel Meshes**, **Center Of Mass**, **Steering Wheel** — см. ту же памятку.

---

## FuelSystem и FuelUI (кратко)

- На машине: **`FuelSystem`** + назначенный **`CarStats`**. Расход и обнуление газа при пустом баке идут из **`CarControl`**.
- На Canvas: **`FuelUI`** → поля **Fuel System** (машина), **Fuel Slider**, опционально текст (legacy или TMP). См. **[Механика транспорт](Механика%20транспорта.md)** → раздел про топливо и HUD.

## CarStats — поля в ScriptableObject (эталон для дизайна)

Файл скрипта: `Assets/_Project/Scripts/Player/Car/CarStats.cs`. Конкретный asset может лежать, например, в `Assets/_Project/` (имя на усмотрение проекта).

Ниже поля **как задумано в коде**. **`maxFuel`** и **`maxSpeedKmh`** использует **`FuelSystem`**; остальное — для таблиц баланса, апгрейдов и будущей связки с **`CarControl`** и другими системами.

### Скорость и разгон (дизайн)

| Поле | По умолчанию | Назначение |
| ---- | ------------ | ---------- |
| **maxSpeedKmh** | 95 | Целевая макс. скорость (км/ч); участвует в формуле расхода топлива в **`FuelSystem`**. |
| **acceleration** | 28 | Ориентир разгона для отображения/апгрейдов. |

### Управляемость и устойчивость (дизайн)

| Поле | По умолчанию | Назначение |
| ---- | ------------ | ---------- |
| **handling** | 0.88 | Множитель управляемости (0.1–1), задумка для будущей связки с рулём. |
| **stability** | 0.42 | Устойчивость / боковая коррекция в аркадной модели (когда будет использоваться). |

### Толпа, топливо, груз, модули

| Поле | По умолчанию | Назначение |
| ---- | ------------ | ---------- |
| **power** | 0.5 | Проходимость сквозь толпу (логика зомби и т.д.). |
| **maxFuel** | 200 | Объём бака для **`FuelSystem`**. |
| **maxCargoWeight** | 5000 | Лимит груза. |
| **moduleSlots** | 3 | Число слотов модулей. |

### Блок «Arcade Drive» в ассете (не используется `CarControl`)

Эти поля остались от старой аркадной схемы без `WheelCollider`; **`CarControl` их не применяет**. Имеет смысл держать для миграции или нового контроллера.

| Поле | Назначение (идея) |
| ---- | ----------------- |
| **driveForce** | Сила тяги (Н). |
| **brakeForce** | Сила торможения (Н). |
| **handbrakeForce** | Сила ручника (Н). |
| **maxTurnRate** | Скорость поворота (°/с). |
| **maxSteerAngle** | Визуальный/логический угол руля (°). |
| **highSpeedSteerFactor** | Ослабление поворота на высокой скорости (0.15–1). |

### Блок «Arcade Grip» и физика

| Поле | Назначение (идея) |
| ---- | ----------------- |
| **lateralGrip** | Резерв под боковую коррекцию / занос в не-WheelCollider модели. |
| **handbrakeGripFactor** | Множитель сцепления при ручнике (0–1). |
| **downforce** | Прижим на скорости (когда появится соответствующая логика на машине). |

## Rigidbody на корне машины (с `CarControl`)

| Параметр | Рекомендация | Комментарий |
| -------- | ------------- | ----------- |
| **Mass** | 1000–2500 | Выше масса — медленнее разгон при том же **Engine Power**, тяжелее толкает зомби. |
| **Drag / Angular Drag** | По умолчанию или умеренно | `CarControl` не выставляет их сам; при «раскручивании» на месте можно чуть поднять **Angular Drag** (осторожно — меняется ощущение). |
| **Interpolate** | Interpolate | `CarControl` выставляет в `Start` — плавнее картинка с камерой. |
| **Collision Detection** | Continuous Dynamic | Тоже задаётся в `Start`. |

Вращение задаётся физикой колёс, не `MoveRotation` из старого контроллера.

## Пресеты «на будущее» (CarStats как ориентир)

Когда свяжешь ассет с машиной, можно ориентироваться на такие соотношения:

### Тяжёлый грузовик

- **maxSpeedKmh** ~50, **maxFuel** высокий, **power** повыше, **handling** ~0.4–0.5  
- На **`CarControl`** сейчас: меньше **Engine Power**, меньше **Turn Speed**, занос слабее (**Drift Amount** ↓, **Drift Sideways Grip Min** ближе к 1).

### Лёгкий пикап

- **maxSpeedKmh** ~80–95, **handling** ~0.75–0.88  
- На **`CarControl`**: **Engine Power** и **Turn Speed** выше, занос чуть сильнее.

### Бронированный фургон

- **maxSpeedKmh** ~45, **maxCargoWeight** / масса выше, **handling** ниже  
- На **`CarControl`**: высокая масса в Rigidbody + умеренный **Turn Speed**.

## Частые проблемы (актуально для CarControl + WheelCollider)

| Проблема | Что проверить |
| -------- | ------------- |
| Не едет / очень медленно | **Engine Power**, масса **Rigidbody**, касаются ли **WheelCollider** земли (слой, высота подвески). Если включён **`FuelSystem`** и бак пуст — газ обнуляется (проверь **`maxFuel`** / расход). |
| Топливо не убывает / UI на месте | **`FuelSystem`** на машине, **`CarStats`** назначен, **`CarControl`** видит **`Fuel System`**, в Play режиме реально жмёшь газ. |
| Шкала топлива пустая в Inspector | В **`FuelUI`** назначены **Fuel System** и **Slider**; при старте сцены **`FuelSystem.Awake`** выставит текущее топливо. |
| Не поворачивает | **Turn Speed**, заполнены ли первые два элемента **Wheels** как передняя ось, не нулевой ли **Horizontal**. |
| Сильно заносит | **Drift Amount** ↓, **Drift Sideways Grip Min** ↑ (ближе к 1), **Drift Speed Ref Ms** ↑. |
| После поворота долго крутит | **Yaw Straighten Strength** ↑, **Steer Return Multiplier** ↑. |
| Дёргается картинка | **Rigidbody Interpolate**, на камере **FollowCamera** — **Smooth Time**, **Max Follow Speed** = 0; цель камеры = корень с физикой. |
| Колёса висят / проваливаются | Радиус **WheelCollider**, **Spring**, позиция относительно меша дороги. |

Подробнее по камере: [Параметры транспорта](Параметры%20транспорта.md) → **FollowCamera**.
