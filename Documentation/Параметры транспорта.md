# Параметры транспорта (Inspector)

Документ описывает поля **`CarControl`**, **`FuelSystem`**, **`FuelUI`** и **`FollowCamera`**. Скрипты: `Assets/_Project/Scripts/Player/Car/CarControl.cs`, `Assets/_Project/Scripts/Player/Car/FuelSystem.cs`, `Assets/_Project/Scripts/UI/FuelUI.cs`, `Assets/_Project/Scripts/Camera/FollowCamera.cs`.

---

## CarControl

Вешается на корень машины с **`Rigidbody`**. Ввод: **Legacy Input Manager** — оси **Vertical** (W/S) и **Horizontal** (A/D).

### Основные поля

| Поле (Inspector) | Тип | Влияние |
| ---------------- | --- | ------- |
| **Engine Power** | float | Момент на колёса при полном газе/заднем ходе (`motorTorque = vertical × enginePower`). Выше — сильнее разгон и толчок назад. |
| **Turn Speed** | float | Максимальный угол поворота передних колёс в градусах. Выше — резче поворот на малых скоростях. |
| **Turn Smoothness** | float | Скорость сглаживания угла руля к целевому значению. Выше — быстрее реакция руля; ниже — плавнее, «мягче» вход в поворот. |
| **Wheels** | Transform[] | Объекты с **`WheelCollider`** (обычно порядок: FL, FR, BL, BR). Первые два индекса получают `steerAngle`, все — моторный момент. |
| **Wheel Meshes** | Transform[] | Визуальные меши колёс; позиция/поворот обновляются из `GetWorldPose` по соответствующему `WheelCollider`. Должны совпадать по порядку с **Wheels**. |
| **Center Of Mass** | Transform | Точка центра масс: в `Start` записывается в `Rigidbody.centerOfMass` как **`localPosition`** этого трансформа относительно корня машины. Влияет на устойчивость и переворот (настраивается вместе с массой и коллайдерами). |
| **Steering Wheel** | GameObject | Опционально: меш руля; крутится по Z пропорционально текущему углу поворота. |

### Drift (subtle)

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Use Drift** | bool | Включено: во время движения с рулением снижается **боковая жёсткость** шин (`sidewaysFriction.stiffness` относительно значения, сохранённого при старте). Выключено: фрикция как на префабе колёс, без динамики заноса. |
| **Drift Amount** | float (0–1) | Общая сила эффекта заноса в формуле смешивания с вводом и скоростью. Выше — сильнее снижение сцепления при тех же условиях. |
| **Drift Sideways Grip Min** | float (0.78–1) | Минимальный множитель боковой жёсткости при «полном» заносе. **Ближе к 1** — меньше скольжение; **ниже** — сильнее занос и дольше инерция боком. |
| **Drift Speed Ref Ms** | float | Ориентировочная планарная скорость (м/с), при которой вклад скорости к полному заносу достигает ~1. **Меньше** — занос заметен раньше (на меньшей скорости); **больше** — сильнее нужно разогнаться, чтобы эффект проявился. |

### Straighten (release steer)

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Steer Input Deadzone** | float | Если \|Horizontal\| не больше этого значения, считается, что руль **отпущен**: включаются гашение поворота машины по Y и ускоренный возврат угла колёс. |
| **Yaw Straighten Strength** | float | При отпущенном руле угловая скорость вокруг вертикальной оси **затухает** экспоненциально за кадр. **Выше** — машина быстрее перестаёт крутиться; **ниже** — дольше сохраняется инерция разворота после заноса. |
| **Steer Return Multiplier** | float (1–4) | Множитель к скорости возврата угла руля к нулю, когда ввод в deadzone. **Выше** — колёса быстрее возвращаются в прямо. |

### Fuel (optional)

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Fuel System** | `FuelSystem` | Если назначен (или найден **`GetComponent<FuelSystem>()`** на том же объекте в `Start`), каждый физический кадр списывается топливо и при пустом баке **газ обнуляется** (моторный момент = 0). Без компонента езда работает как без топлива. |

### Поведение, заданное в коде (не в Inspector)

- В **`Start`** на `Rigidbody` выставляются **Interpolate** и **Continuous Dynamic** — для плавной картинки при следовании камеры.
- Занос не меняет **переднюю** фрикцию и не режет скорость отдельной логикой — только боковое сцепление шин.

## FuelSystem

Вешается на корень машины (рядом с `CarControl`). Читает **`CarStats`**.

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Stats** | `CarStats` | Обязательно для старта: **`maxFuel`** — полный бак в начале заезда и верхняя граница для **`Refuel`**; **`maxSpeedKmh`** — нормализация скорости в расходе. |
| **Base Consumption Per Second** | float | Базовый расход в секунду при полном газе (до добавок от скорости). |
| **Speed Consumption Factor** | float | Дополнительный расход, масштабируемый отношением текущей скорости к **`maxSpeedKmh`**. |

События: **`OnFuelChanged(current, max)`**, **`OnFuelEmpty`**.

## FuelUI

Вешается на объект под **Canvas** (например панель HUD).

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Fuel System** | `FuelSystem` | Ссылка на топливо машины (игрока). |
| **Fuel Slider** | `Slider` | Минимум/максимум и значение синхронизируются с топливом. |
| **Fuel Text** | legacy `Text` | Опционально: строка `текущее/макс`. |
| **Fuel Text TMP** | `TextMeshProUGUI` | То же для TMP (достаточно одного из двух текстов). |

## FollowCamera

Вешается на **Main Camera** (или отдельный объект камеры). Цель — трансформ машины (корень с `Rigidbody`).

### Target

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Target** | Transform | Объект, за которым следует камера (обычно корень пикапа). Если пусто, при старте ищется объект с тегом **Player**. |

### Position

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Offset** | Vector3 | Смещение в **локальном пространстве цели**: X — вправо, Y — вверх, **отрицательный Z** — сзади относительно перед машины. Задаёт дистанцию и высоту обзора. |
| **Look Ahead Distance** | float | Дополнительный сдвиг точки камеры вперёд по `target.forward` — больше дороги перед машиной в кадре. |

### Smoothing

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Smooth Time** | float (0.05–0.8) | Время сглаживания позиции (`SmoothDamp`). **Больше** — мягче, меньше дёрганий при рывках физики; **меньше** — камера плотнее к цели, отзывчивее. |
| **Max Follow Speed** | float | Максимальная скорость движения камеры к цели (ед/с). **0** — без ограничения. Слишком маленькое значение может давать ступенчатое следование при резком разгоне. |
| **Rotation Speed** | float (1–20) | Как быстро камера поворачивает взгляд к точке смотрения. **Выше** — меньше отставание при поворотах машины. |

### Mouse orbit (vehicle doc)

| Поле | Тип | Влияние |
| ---- | --- | ------- |
| **Mouse Orbit Enabled** | bool | Включено: горизонтальный поворот камеры вокруг машины по оси **Mouse X** (накопительный угол `_orbitYaw`). |
| **Mouse Orbit Sensitivity** | float (0.1–8) | Множитель чувствительности мыши по горизонтали. |
| **Lock Cursor In Play Mode** | bool | При старте сцены курсор скрывается и блокируется (как в шутерах); удобно для теста, может мешать в редакторе. |

### Поведение, заданное в коде

- Желаемая позиция: `target.TransformPoint(rotation(Mouse) * offset) + target.forward * lookAheadDistance`.
- Позиция сглаживается через **`Time.smoothDeltaTime`**, чтобы уменьшить микродёргания при нестабильном FPS.

## Связь с дизайн-таблицей (CarStats)

Поля из **`CarStats`** описывают целевые характеристики транспорта. **`FuelSystem`** использует **`maxFuel`** и **`maxSpeedKmh`**. **`CarControl`** по-прежнему не читает ассет для мотора и руля — **Engine Power**, **Turn Speed** и занос настраиваются в Inspector у `CarControl` (или будущая обёртка подставит значения из `CarStats`).
