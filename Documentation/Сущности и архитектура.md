# **2\. Zombie**

### **Компоненты (целевая архитектура):**

* **ZombieAI** — состояния, активация по радиусу, цель = игрок  
* **ZombieStats** (ScriptableObject)  
* **ZombieMovement** — движение по **NavMesh**, обход препятствий и других зомби (настройка агента / избегание)  
* **ZombieCrowdResistance** *(или логика внутри AI / отдельный helper)* — вклад в замедление машины при контакте; агрегация «один vs толпа»  
* **ZombieDeathHandler** — смерть; **сейчас заглушка** без регдолла  
* **ZombieLODHandler** — оптимизация на дистанции  

**Отложено (не подключается до отдельной задачи «цепляние»):**

* **ZombieAttachBehaviour** — до **5** висящих на авто; связка с **ZombieAttachmentSystem** на машине  

**Отложено (визуал):**

* **ZombieRagdollController** — когда появится регдолл; до тех пор не используется или stub  

Типы:

* NormalZombie  
* HeavyZombie  
* BossZombie
