## Статус прототипа (кратко)

- Движение: **`CarControl`** + **`Rigidbody`** + **4× `WheelCollider`** (модель пикапа — `Assets/Models/Pack_Pickup`).
- Ввод: **W / S / A / D / Space** (частично прямой опрос клавиш в `CarControl`; оси Legacy **Horizontal** / **Vertical** не обязательны для базового WASD).
- Топливо: **`FuelSystem`** + **`CarStats`**, расход из `CarControl`; UI — **`FuelUI`**.
- HP: **`VehicleHealth`**; при 0 HP газ отключается; урон от зомби — см. механику зомби.
- Толпа: **`VehicleCrowdBrake`** по **`CarStats`**.

## Подвеска — требуется донастройка

Поведение на неровностях (брёвна, кочки) сильно зависит от **базовых** значений `WheelCollider` (ход подвески, пружина/демпфер в инспекторе) и от множителей в **`CarControl` → Suspension Stiffness**.

- В **`Start`** скрипт **сохраняет базу** с каждого колеса и применяет множители **от этой базы** (без накопления при перезапусках).
- Порядок пер-колёсных массивов: **FL, FR, BL, BR** — `perWheelSuspensionDistanceScale`, `perWheelSpringScale`, `perWheelDamperScale`.
- **`maxDepenetrationVelocity`** на `Rigidbody` задаётся из `CarControl` — меньше значение обычно сглаживает «выталкивание» из геометрии.

**Рекомендация:** после изменения уровня (новые препятствия) прогнать Play Mode и подстроить мягкость/ход; цель — небольшая раскачка кузова и контакт всех колёс без сильных подпрыгиваний.

## CarControl — основные поля

Вешается на корень машины с **`Rigidbody`**. Тег **`Player`** нужен зомби для контакта.

| Поле | Назначение |
|------|------------|
| **Engine Power** | Момент на колёса (через коробку-автомат в коде). |
| **Turn Speed** | Макс. угол поворота передних колёс (градусы). |
| **Turn Smoothness** | Зарезервировано/наследие; основной вход руля — ramp (ниже). |
| **Max Speed Kmh** | Потолок скорости по плоскости (0 = выкл). |
| **Wheels** | Трансформы с **`WheelCollider`** (обычно FL, FR, BL, BR). |
| **Wheel Meshes** | Визуал колёс; **не те же объекты**, что коллайдеры. |
| **Center Of Mass** | `localPosition` пишется в `Rigidbody.centerOfMass`, `automaticCenterOfMass = false`. |
| **Steering Wheel** | Опционально: меш руля. |

### Drift

| Поле | Назначение |
|------|------------|
| **Use Drift** | Динамическое снижение боковой жёсткости (перед/зад раздельно). |
| **Drift Amount** | Сила смешивания заноса. |
| **Drift Rear Grip Min** / **Drift Front Grip Min** | Мин. множители боковой жёсткости зад/перед при полном заносе. |
| **Drift Speed Ref Ms** | Ориентир скорости (м/с) для вклада скорости в занос. |
| **Drift Entry Boost**, **Drift Speed Entry Multiplier** | Вход в занос. |
| **Drift Turn Speed Limit 01** … **Drift Reverse Recover Strength** | Помощник высокоскоростного дрифта: дуга, ограничение yaw, анти-180, удержание скорости при газе. |

### Торможение и коастинг

**Brake Power**, **Release Brake Power**, **Handbrake Power**, **Handbrake Rear Grip**.  
**Coast:** линейное замедление при отпускании газа (**Use Constant Coast Decel**, **Coast Deceleration Mps2**), **Coast Lateral Stabilize** при отпущенном руле.

### Подвеска (множители к базе на WheelCollider)

**Suspension Spring Multiplier**, **Suspension Damper Multiplier**, **Suspension Distance Multiplier**, **Wheel Damping Rate Multiplier**, **Suspension Target Position Offset**, **Force App Point Distance**, пер-колёсные масшивы (см. выше). **Target Vehicle Mass** — масса `Rigidbody`. **Pitch Stabilization**, **Max Depenetration Velocity**.

### Прочее

**Grip Profiles** — слои поверхности и множители forward/sideways. **Transmission** — упрощённая автоматическая КПП (передаточные числа, RPM). **Steering Ramp**, **Steering Weight**, **Steer Axis Noise Deadzone**. **Straighten** — мёртвая зона руля, гашение yaw, возврат руля. **Throttle Rise / Fall**. **Effects / Audio** — клипы, громкости, следы (**Tire Mark Material**), дым/трейлы при проскальзывании.

## CarStats (ScriptableObject)

Файл: `Assets/_Project/Scripts/Player/Car/CarStats.cs`. В езде напрямую из ассета читают **`FuelSystem`** (**maxFuel**, **maxSpeedKmh**) и **`VehicleCrowdBrake`**. Остальные поля — баланс и будущие связки. Блоки Arcade Drive/Grip в ассете могут не использоваться `CarControl` — см. код.

## Rigidbody (корень машины)

`CarControl` в `Start` выставляет **Interpolate**, **Continuous Dynamic**, массу, **maxDepenetration velocity**, увеличивает **solver iterations**. Ручная настройка **Drag** в основном не требуется из-за коастинга в коде.

## Пресеты «на будущее» (ориентир)

- **Тяжёлый грузовик:** ниже **Engine Power**, **Turn Speed**, слабее занос; выше масса.  
- **Лёгкий пикап:** выше скорость и отзывчивость руля, чуть сильнее занос.  
Согласуй **maxSpeedKmh** в `CarStats` с **Max Speed Kmh** в `CarControl`.

## Частые проблемы

| Симптом | Проверить |
|---------|-----------|
| Не едет | **Engine Power**, масса, касание земли колёсами, топливо/HP. |
| Не поворачивает | **Turn Speed**, первые два **Wheels** — передняя ось. |
| Сильный занос / разворот | **Drift Amount**, grip min, блок high-speed drift assist. |
| Дёргается картинка | Interpolate на RB, настройки **FollowCamera**. |
| Прыжки на препятствиях | Подвеска (база на колесе + множители в `CarControl`), **CarPhysic** (bounce 0), **Max Depenetration Velocity**. |
| Коллайдер кузова | Должен быть с материалом без отскока; не заменять визуалом колёс `WheelCollider`. |
